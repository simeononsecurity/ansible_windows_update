# Ansible Role: windows_update

This role automates Windows security patching using Ansible.

## Requirements

- Windows hosts
- Ansible 2.10 or later

## Role Variables

None

## Dependencies

None

## Installation
```bash
ansible-galaxy install simeononsecurity.windows_update
```

## Example Playbook

```yaml
- name: Apply Windows updates
  hosts: windows_hosts
  become: yes

  roles:
    - windows_update
```
